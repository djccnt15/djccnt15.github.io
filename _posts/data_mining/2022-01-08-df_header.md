데이터를 다루다보면 아래 예시 데이터프레임과 같이 칼럼명이 대문자로 되어 있는 경우들이 있다.  

```python
import pandas as pd

a = {'A':[1, 2, 3], 'B':[2, 4, 6]}
d = pd.DataFrame.from_dict(a)

print(d)
```
```markdown
   A  B
0  1  2
1  2  4
2  3  6
```

대소문자를 왔다갔다 하면서 코딩하려면 매우 귀찮은데, 이럴 땐 아래와 같이 `lambda` 함수를 이용해서 한번에 모두 소문자로 바꿔줄 수 있다.

```python
d.columns = map(lambda x: x.lower(), d.columns)

print(d)
```
```markdown
   a  b
0  1  2
1  2  4
2  3  6
```
