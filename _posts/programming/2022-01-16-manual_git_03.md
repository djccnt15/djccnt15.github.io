원격 저장소에서 파일을 가져오기 위한 명령어들

## 1. pull
원격 저장소의 내용을 가져와서 현재 브랜치와 병합(merge)하는 명령어

```powershell
$ git pull [repository_name] [branch_name]
```

## 2. clone
로컬 저장소의 내용을 원격 저장소의 내용과 일치시키는 명령어로, 폴더의 작업 내용이 사라지기 때문에 프로젝트를 처음 시작할 때만 사용한다.

```powershell
$ git clone <url>
```
