## MkDocs 서버 구동 명령어

주로 사용하는 서버 구동 옵션은 아래와 같다.  

```bat
mkdocs serve
```

`--host`, `--port` 옵션을 사용하지 않을 경우 기본 IP 및 포트는 아래와 같이 할당된다.  

- [http://127.0.0.1:8000](http://127.0.0.1:8000)
- [http://localhost:8000](http://localhost:8000)
