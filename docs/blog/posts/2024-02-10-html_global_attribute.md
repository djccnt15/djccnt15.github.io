## style

태그의 스타일을 입력하기 위한 속성.  

!!! tip
    태그의 스타일은 별도의 CSS 파일로 분리하는 것이 좋다.  

## class

태그를 지칭하는 **중복 가능**한 이름  

## id

태그를 지칭하는 **중복 불가능**한 이름
