주어진 다이얼을 돌리는데 필요한 최소 시간을 구하라.  

<!-- more -->

---

다이얼이 되돌아가는데 걸리는 시간은 없는 것으로 되어 있으니, 아래와 같이 인덱스 메서드를 활용해서 쉽게 풀이할 수 있다.  

```python
d = ['ABC', 'DEF', 'GHI', 'JKL', 'MNO', 'PQRS', 'TUV', 'WXYZ']
print(sum(d.index(b) + 3 for a in input() for b in d if b.find(a) >= 0))
```
